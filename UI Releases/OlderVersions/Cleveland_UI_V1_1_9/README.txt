-------------------------------------------------
Cleveland UI T1 skillet. Version 1.1.8 Creating date 2024-04-22
-------------------------------------------------

Steps to install package:
1- On USB stick create folder named welbilt
2- Inside the welbilt folder create folder named package
3- Put the released package for example Cleveland_V1_1_9.tar inside the package folder.
4- Insert the USB stick on the USB port for the UI.
5- After 3 to 4 minutes, a screen will appear indicating completion of system update and system will reboot.
 
Note: 
- Make sure USB stick is FAT32 formatted. 
- Remove USB once upload completed.
